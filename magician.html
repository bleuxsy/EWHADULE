에타 마법사란?

수강신청을 실패할 가능성을 염두에 뒤 Plan B 시간표를 쉽게 만들어주는 에브리타임 기능

원하는 과목을 그룹마다 1개 이상 담으면 각 그룹에서 과목을 1개씩 꺼내 경우의 수를 조합하여 만들어질 수 있는 모든 시간표를 보여준다.

1. 시간표 탭으로 이동

에브리타임 홈 화면 하단에 있는 메뉴 중 왼쪽 두번째에 위치한 '시간표'를 눌러 시간표 화면으로 이동한다.

2. 시간표 마법사 실행

새 시간표 만들기를 통해 시간표를 만들어준다.

만들어진 시간표 상단의 우측 상단 첫번째 메뉴인 '+' 버튼을 누른다.

화면 우측 상단의 첫번째 메뉴인 '마법사' 버튼을 누른다.

3. 마법사로 수업 구성

각 그룹에 있는 '+' 버튼을 통해 그룹을 구성한다.

tip!) 시간표 마법사는 각 그룹에 넣어둔 강의들을 1개씩 선택하여 구성할 수 있는 모든 시간표의 경우의 수를 보여주기 때문에 학수번호와 강의 시간이 겹치는 강의들은 각 그룹에 1개만 넣어주는게 좋다.

그룹을 강의들로 구성한 뒤 우측 상단에 위치한 '결과 보기' 버튼을 누른다.

4. 시간표 저장

시간표 마법사로 만들어진 시간표들 중 마음에 드는 시간표를 골라 저장해준다.

5. 시간표 수정

시간표 마법사로 완성된 시간표들 중 가장 마음에 드는 시간표로 더 세부적으로 수정해준다.
